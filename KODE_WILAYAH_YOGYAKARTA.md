# Kode Wilayah DIY Yogyakarta untuk API BMKG

Dokumen ini berisi kode wilayah administrasi tingkat IV (adm4) untuk Daerah Istimewa Yogyakarta yang digunakan untuk API Prakiraan Cuaca BMKG.

## üìç Struktur Kode Wilayah

Format: `XX.XX.XX.XXXX`
- 2 digit pertama: Kode Provinsi (34 = DIY Yogyakarta)
- 2 digit kedua: Kode Kabupaten/Kota
- 2 digit ketiga: Kode Kecamatan
- 4 digit terakhir: Kode Kelurahan/Desa

## üîç Cara Mencari Kode Wilayah Lainnya

Untuk mendapatkan kode wilayah lengkap, Anda dapat:

1. **Mengakses Portal Data BMKG**: https://data.bmkg.go.id/prakiraan-cuaca/
2. **Menggunakan Keputusan Menteri Dalam Negeri**: Nomor 100.1.1-6117 Tahun 2022
3. **Mencari di Situs Resmi Kemendagri**: https://www.kemendagri.go.id/

## üìù Contoh Penggunaan

### PHP
```php
// Kota Yogyakarta - Gondomanan
$kodeWilayah = '34.71.01.1001';
$url = "https://api.bmkg.go.id/publik/prakiraan-cuaca?adm4=" . $kodeWilayah;
```

### JavaScript
```javascript
// Sleman - Depok
const kodeWilayah = '34.04.07.1001';
const url = `https://api.bmkg.go.id/publik/prakiraan-cuaca?adm4=${kodeWilayah}`;
```

## ‚ö†Ô∏è Catatan

1. **Kode Kelurahan vs Desa**: 
   - Kelurahan biasanya dimulai dengan `1001`, `1002`, dst.
   - Desa biasanya dimulai dengan `2001`, `2002`, dst.

2. **Validasi Kode**: 
   - Pastikan kode wilayah valid sebelum digunakan
   - Test dengan API BMKG untuk memastikan kode bekerja

3. **Update Kode**: 
   - Kode wilayah dapat berubah jika ada pemekaran wilayah
   - Selalu refer ke sumber resmi terbaru
